# Ignite UI for Blazor

Welcome to the **Ignite UI for Blazor** component library stories.

Browse the sidebar to explore each component with interactive demos and documentation.

## Getting Started

Install the NuGet package:

```bash
dotnet add package IgniteUI.Blazor.Lite
```

Register the services in your `Program.cs`:

```csharp
builder.Services.AddIgniteUIBlazor();
```

Add the required stylesheet and script to your `App.razor`:

```html
<link rel="stylesheet" href="_content/IgniteUI.Blazor/themes/light/bootstrap.css" />
<script src="_content/IgniteUI.Blazor/app.bundle.js"></script>
```
